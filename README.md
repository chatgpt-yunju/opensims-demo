## 源码使用说明

### 1. 环境准备

```bash
pip install -r requirements.txt
```

建议使用 Python 3.10+。

### 2. 配置火山方舟 API Key

不要把密钥写入源码或提交到 Git。使用环境变量：

```powershell
$env:ARK_API_KEY="your_volcengine_ark_api_key"
```

当前默认模型配置位于 `config.py` 和 `settings.py`：

```python
API_ENDPOINT = "https://ark.cn-beijing.volces.com/api/v3/responses"
API_MODEL = "deepseek-v3-2-251201"
USE_MOCK = False
```

如果没有配置 `ARK_API_KEY`，或接口调用失败，程序会降级到内置 Mock 回复，保证本地功能可用。

### 3. 启动源码 GUI

```bash
python gui_simple.py
```

### 4. 运行核心测试

```bash
python test_social_auto_chat_scheduler.py
```

该测试会验证 Smallville 风格自动社交调度，包括：

- 自动选择对话对象
- 控制对话轮数
- 写入社交记忆
- 防止短时间重复打扰

### 5. 打包 exe

```bash
python build_gui_simple.py
```

最近一次验证生成的 exe：

```text
dist/OpenSims_GUI_Simple_1.0.3.27.exe
```

### 6. 关键文件

| 文件 | 说明 |
| --- | --- |
| `api_client.py` | 火山方舟 DeepSeek / OpenAI-compatible API 调用封装 |
| `auto_chat_scheduler.py` | Smallville 风格自动社交调度器 |
| `virtual_human.py` | 仿人智能体状态、记忆、关系模型 |
| `gui_simple.py` | 桌面 GUI 入口 |
| `config.py` | 默认模型、自动聊天和系统配置 |
| `settings.py` | GUI 设置文件管理 |
| `test_social_auto_chat_scheduler.py` | 自动社交调度验证脚本 |
| `MULTI_AGENT_DEEPSEEK_DIALOGUE_REPORT.md` | 最新多智能体 DeepSeek 对话报告 |
