# OpenSims v1.0.2 发布说明

**发布日期**: 2025-04-05
**版本**: 1.0.2.1
**文件**: `dist/OpenSims_v1.0.2.exe`

## ✨ 新功能

### 1. Multi-Agent 协作引导系统
每个导师虚拟人内置5个专业角色，协同生成引导回复：
- **规划师**：制定探索框架和步骤
- **反驳师**：挑战假设，避免盲目决策
- **反馈师**：提供具体可操作建议
- **鼓励师**：情感支持和动机强化
- **整合师**：综合多角度观点形成最终回复

不同导师类型偏好不同输出风格：
- 创意激发者 → 活泼、带Emoji
- 目标规划师 → 结构化、分步骤
- 情感支持者 → 温暖、共鸣

### 2. 导师主动触发引擎
教师和贵人在适当时机主动找玩家：
- 🎯 **触发条件**：
  - 收集≥3条使命线索但未整合
  - 完成挑战后需要复盘
  - 发现兴趣但无行动
  - 周期性主动关怀（15%概率）
- 🎲 **导师选择算法**：关系度 + 专长匹配 + 随机平衡
- ⏱️ **冷却保护**：同一导师24小时内不重复触发，每天最多2次

### 3. 玩家成长追踪系统
完整记录玩家探索轨迹：
- 发现兴趣领域
- 技能熟练度
- 使命线索碎片（自动检测关联）
- 人生事件时间线
- 挑战完成数
- 引导历史（每次对话记录）

### 4. 引导日志查看功能
游戏中按选项 **"2. 📊 查看成长轨迹"** 即可查看：
- 当前使命状态
- 成长指标（兴趣、挑战、线索、引导次数）
- 技能熟练度Top 5（可视化进度条）
- 最近10条引导记录
- 所有使命线索及关联分析
- 人生事件时间线
- 导师关系度面板

### 5. 关系度动态系统
- 每次对话增加关系度（+2，高质量回复+3）
- 导师主动引导额外增加（+2）
- 关系度影响引导效果（>80时效果+30%）
- 虚拟人之间也有关系网络（影响自动对话内容）

## 🎮 界面优化

主菜单重新设计，**导师引导为核心**：

```
  1. 💬 与导师对话（核心功能）
  2. 📊 查看成长轨迹
  3. 👥 查看所有角色
  4. 🔄 切换当前角色
  5. ➕ 创建AI角色
  6. ⏳ 结束今天
  7. ⚙️  更多选项（模拟人生行动）
  8. 🚪 退出游戏
```

次要功能（吃饭、睡觉、工作、购物、小红书发布等）收纳到 **"7. 更多选项"** 子菜单。

## 🔧 技术细节

### 新增文件
- `player_growth.py` - 玩家成长数据模型
- `multi_agent_guidance.py` - Multi-Agent协作引擎
- `mentor_trigger.py` - 主动触发引擎
- `build_info.py` - 构建信息（自动生成）
- `test_all_systems.py` - 集成测试
- `test_guidance_log.py` - 引导日志测试

### 修改文件
- `virtual_human.py` - 导师属性、关系度更新方法、职业状态同步
- `main.py` - 集成成长系统、触发引擎、关系度更新、新菜单
- `auto_chat_scheduler.py` - 社交网络效应（虚拟人讨论玩家进展）
- `config.py` - 添加"贵人"职业
- `version_manager.py` - 版本管理

### 关键改进
- 所有对话自动更新关系度
- 导师职业=教师/贵人时自动标记 `is_mentor=True`
- 职业变更时自动更新导师状态
- 自动聊天调度器传递玩家成长上下文
