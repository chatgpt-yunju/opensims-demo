# OpenSims v1.0.2 - 快速开始

## 运行游戏

### 方式1：直接运行exe（推荐）
```bash
双击运行: dist/OpenSims_v1.0.2.exe
# 或使用启动器
start_opensims_v1.0.2.bat
```

### 方式2：从源码运行
```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python main.py
```

## 首次使用

### 1. 创建角色
首次运行会提示创建虚拟人：
- 输入昵称（如"小明"）
- 选择性格：温柔型/幽默型/严肃型/中立型
- 标记为**玩家角色**（输入y）

### 2. 添加导师
导师会在你探索过程中主动引导：
- 创建AI角色后，在"更多选项"中找工作
- 反复尝试直到职业变为 **"教师"** 或 **"贵人"**
- 或直接编辑存档修改 `job` 字段

### 3. 开始对话
主菜单选择 **"1. 💬 与导师对话"**，输入你想聊的内容：
```
"我想找到人生使命，有什么建议吗？"
"我对编程感兴趣，但犹豫是否深入"
"最近感到很迷茫..."
```

导师会基于你的成长状态给出个性化引导。

### 4. 查看成长轨迹
随时按 **"2. 📊 查看成长轨迹"** 查看：
- 已发现兴趣领域
- 使命线索收集进度
- 挑战完成情况
- 导师关系度
- 完整引导历史

## 特色系统

### Multi-Agent 协作
每个导师内置5个专业角色协同工作：
- 规划师 → 制定探索路径
- 反驳师 → 挑战假设
- 反馈师 → 提供建议
- 鼓励师 → 情感支持
- 整合师 → 综合输出

### 导师主动触发
导师会在以下情况主动来找你：
- ✅ 使命线索需要整合（≥3条）
- ✅ 完成挑战后需要复盘
- ✅ 发现兴趣但未实践
- ✅ 周期性主动关怀

### 关系度系统
- 每次对话 +2关系度
- 高质量回复额外 +1
- 导师主动引导 +2
- 高关系度（>80）提升引导效果30%

## 配置文件

编辑 `config.py` 调整：
```python
USE_MOCK = True  # True使用Mock回复（无需API），False使用真实AI
API_KEY = ""     # 你的API密钥
API_ENDPOINT = "https://api.yunjunet.cn/v1/chat/completions"
```

## 常见问题

### Q: 导师不主动找我？
A: 确保导师职业=教师/贵人，且关系度≥30。主动触发受冷却限制（每天最多2次）。

### Q: 如何让对话更智能？
A: 配置 `USE_MOCK = False` 并填入 `API_KEY`，即可使用真实AI（推荐Step或Claude）。

### Q: 群聊在哪里？
A: 在"更多选项" → "群聊"，选择至少3个角色。

### Q: 如何查看所有对话历史？
A: 查看成长日志 → "最近引导记录"（最近10条）。

### Q: 可以导出成长数据吗？
A: `demo_data.json` 包含所有数据，`player_growth` 字段即成长数据。
